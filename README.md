# Sabor Salvadoreño - Landing Page Gastronómica

Landing page interactiva dedicada a promover la gastronomía típica de El Salvador. Proyecto evaluado para Transformación Digital.

## 🚀 Demo En Vivo
[Ver sitio web en producción](https://github.com/jhonitorres-coedsantiago/SaborSalvadore-o.git)

## 🛠️ Tecnologías Utilizadas
* HTML5 (Estructura semántica)
* CSS3 (Flexbox, CSS Grid, Media Queries para Responsividad)
* JavaScript Vanilla (Manipulación del DOM, Regex, Eventos)

## ✨ Funcionalidades
* **Navegación Típica:** Navbar fija con cambio de estilo al hacer scroll y menú hamburguesa responsivo.
* **Menú con CSS Grid:** Tarjetas de platillos tradicionales (Pupusas, Tamales, Antojitos) adaptables a cualquier pantalla.
* **Formulario de Reservas/Contacto:** Validación avanzada en tiempo real con expresiones regulares (Regex), limpieza de errores y mensaje de éxito personalizado.
